# STUDIO-BRIEF-TEMPLATE.md — Standard Input For Generating A Studio Template

This is the standard input every AI tool consumes when generating a studio website template variant. Every section is required. If a section genuinely does not apply, write `N/A` with a one-line reason. Do not delete sections.

The completed brief is what sets the ceiling on quality across all variants. Tight, specific briefs produce tight, specific templates. Vague briefs produce generic ones.

Below the blank template you will find a fully worked example for a fictional studio (Rivertown Dance Co) so you can see what a completed brief looks like.

## Worked Example

Below is the same brief filled in for a fictional studio. Use it as a calibration reference for the level of detail expected.

### Studio Name

Rivertown Dance Co

### Location

Knoxville, Tennessee, United States

### Existing Website URL

`https://rivertowndance.example.com` (built on a Wix template around 2019, not mobile-optimised, no booking)

### Studio Size

- Active students: 412
- Locations: 1 (Knoxville, Bearden neighbourhood)

### Studio Vibe

Contemporary, urban, polished, community-minded, energetic.

### Target Audience

- [ ] Parents of young dancers (preschool, kinder, early primary)
- [x] Parents of school-age recreational dancers
- [x] Teen recreational dancers booking for themselves
- [x] Competitive dance families (full programme commitment)
- [x] Adult recreational dancers (fitness, social, hobby)
- [ ] Pre-professional / vocational students
- [ ] Other: N/A

The mix is roughly 55% school-age recreational, 30% competitive, 15% adult drop-in. Adults are growing fastest and the studio wants the homepage to reflect that without burying the competitive programme.

### Dance Styles Offered

Hip hop, jazz, contemporary, ballet (recreational and pre-pointe), tap, acro, adult heels, adult conditioning.

### Key Pages Needed

- [x] About
- [x] Classes / programmes
- [x] Schedule
- [x] Instructors / faculty
- [ ] Recital info (handled via newsletter, not on the marketing site)
- [x] Parent portal link
- [x] Contact
- [x] Trial booking
- [x] Enrollment / registration
- [x] FAQ
- [ ] Policies (linked from FAQ, no standalone page)
- [ ] News / blog
- [ ] Other: N/A

### Brand Colours And Imagery Notes

- Existing brand colours: charcoal `#1F1F23`, off-white `#F4EFE6`, accent terracotta `#C8553D`. Confident in these.
- Logo status: refreshed in 2024, wordmark only, supplied as SVG.
- Photography style: their own in-studio shots from a 2024 session. Natural light, dancers in motion, faces visible, mix of solo and group. Mid-warm grade, not high-contrast.
- Anything they have explicitly said about look and feel: "We want to look like a studio in Brooklyn or LA, not a typical Southern dance school. Sophisticated, not flashy."

### Special Features Needed

- [x] Online enrollment
- [x] Trial class booking
- [x] Parent portal integration / link
- [ ] Recital info display
- [x] Instructor bios
- [x] Class schedule with filtering (by age, by style, by day)
- [ ] Pricing display (handled at trial booking, not on marketing pages)
- [x] Newsletter signup
- [ ] Studio policies download
- [ ] Other: N/A

### Reference Sites They Like

1. `https://broadwaydancecenter.com` — clean, faculty-led credibility, strong instructor cards.
2. `https://steezy.co` — modern, motion-led hero, friendly tone for adult drop-ins.
3. `https://www.alvinailey.org/school` — sophisticated typography, restrained colour, serious about craft.

### Anything To Avoid

- Pink and purple gradients. They read as "kids studio" and the owner has been explicit she does not want that read.
- Tutus and ballet-pointe-shoes-only imagery in the hero. They want their range visible immediately.
- Hard-sell competition language ("champions", "elite squad"). The competitive programme exists, but the studio's positioning is community and craft, not trophies.
- Stock photography of any kind on landing pages.

Gary, StudioLAB founder.
