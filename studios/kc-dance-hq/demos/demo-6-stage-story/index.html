Before you book

Clear answers keep momentum.

Can my child try a class first?

Yes. The first class is free, which gives your family a low-pressure way to see whether the class feels right.

Is there a class for beginners?

Yes. KCDanceHQ offers pathways for preschoolers, children, teens and adults, with options for different confidence levels.

Where is the full timetable?

The current timetable belongs on the dedicated timetable page so families can see the latest class details in one place.

Do students perform?

Students have the opportunity to participate in the annual production, which turns weekly learning into a memorable stage experience.
