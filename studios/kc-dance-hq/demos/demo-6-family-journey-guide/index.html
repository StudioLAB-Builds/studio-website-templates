Questions before the trial

Reduce the mental load for families.

Good conversion here is reassurance. The page should answer common concerns without creating fake urgency.

Can we start if my child has never danced?

Yes. The enquiry asks for age and experience so the team can recommend a suitable class.

Do families need to join competitions?

No. KCDanceHQ is positioned around quality recreational dance education, not compulsory competition pathways.

What should a new dancer wear?

The homepage can keep this simple and direct families to the info pack, where class-specific details can be confirmed.

Where is the timetable?

The homepage should link to a dedicated live timetable page instead of showing static schedule data that may go stale.
